# Amazon Sales Analytics

This project analyzes Amazon sales data to uncover trends, patterns, and insights.  
It helps understand customer behavior, product performance, and overall sales dynamics.

## 📂 Project Structure
- **src/** → Source code for data processing and analysis  
- **data/** → Raw and processed datasets  
- **charts/** → Visualizations and plots  
- **notebooks/** → Jupyter notebooks for exploration  
- **outputs/** → Final reports and results  

## 🚀 Features
- Data cleaning and preprocessing  
- Exploratory data analysis (EDA)  
- Sales trend visualization  
- Product performance insights  
- Exportable charts and reports  

## 🛠️ Technologies Used
- Python (pandas, numpy, matplotlib, seaborn)  
- Jupyter Notebook  
- SQLite / JSON
